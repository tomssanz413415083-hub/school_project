# Indonesian Culture Presentation (PPT)


## Project Introduction
This project is created to present and introduce the richness of Indonesian culture.
It is part of an individual assignment focusing on cultural understanding and presentation skills using slides.


## Project Description
This presentation covers various aspects of Indonesian culture, including historical sites, traditional music, dances, and arts.

The topics included are:
1. Borobudur Temple (Candi Borobudur)
2. Tirta Empul Temple (聖泉寺)
3. Sape (Traditional Musical Instrument)
4. Gamelan (Traditional Ensemble Music)
5. Sasando (Traditional String Instrument)
6. Tari Kecak (Traditional Dance)
7. Tari Banaspati (Traditional Dance)
8. Batik (Traditional Textile Art)
9. Wayang Kulit (Shadow Puppet Theatre)

The presentation is designed using a question-based format to encourage discussion and deeper understanding of each cultural element.


## Features of the Presentation
* Bilingual content (Chinese & Indonesian/English)
* Cultural explanations through guided questions
* Covers architecture, religion, music, dance, and art
* Visual and structured slide format for easy understanding


## Purpose
* To improve knowledge about Indonesian culture
* To enhance presentation and communication skills
* To explore the relationship between culture, religion, and society
